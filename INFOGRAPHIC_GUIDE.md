# 🎨 Гайд по Созданию Инфографики

## 🛠️ Рекомендуемые Инструменты

### 1. **Gamma.app** (Лучший выбор для презентаций)
**Почему:**
- AI-генерация слайдов из текста
- Современный дизайн
- Быстрое создание
- Интерактивные элементы

**Как использовать:**
1. Зайти на gamma.app
2. Создать новую презентацию
3. Выбрать "Import from text"
4. Скопировать содержимое из `GAMMA_PRESENTATION_CONTENT.md`
5. AI создаст красивую презентацию
6. Отредактировать детали

**Цена:** Бесплатно (с лимитом) или $10/месяц

---

### 2. **Figma** (Для профессиональной инфографики)
**Почему:**
- Полный контроль над дизайном
- Векторная графика
- Совместная работа
- Экспорт в любой формат

**Шаблоны для старта:**
- Figma Community → "Bot Flow Chart"
- Figma Community → "User Journey Map"
- Figma Community → "Infographic Template"

**Как создать:**
1. Создать новый файл
2. Использовать Auto Layout для блоков
3. Добавить иконки из Iconify
4. Соединить блоки стрелками
5. Экспортировать в PNG/PDF

---

### 3. **Canva** (Простой и быстрый)
**Почему:**
- Готовые шаблоны
- Drag & drop интерфейс
- Не требует навыков дизайна
- Большая библиотека элементов

**Шаблоны:**
- "Process Infographic"
- "Flowchart"
- "Business Presentation"

**Как создать:**
1. Выбрать шаблон "Infographic"
2. Заменить текст на свой
3. Добавить иконки для кнопок
4. Настроить цвета под бренд
5. Скачать в PDF/PNG

---

### 4. **Draw.io / Diagrams.net** (Для блок-схем)
**Почему:**
- Бесплатно
- Специализация на схемах
- Простой интерфейс
- Экспорт в разные форматы

**Как создать:**
1. Открыть draw.io
2. Выбрать "Blank Diagram"
3. Использовать блоки из левой панели
4. Соединить стрелками
5. Экспортировать

---

### 5. **Miro** (Для интерактивных карт)
**Почему:**
- Бесконечный холст
- Совместная работа
- Интерактивность
- Шаблоны для User Journey

**Как создать:**
1. Создать новую доску
2. Выбрать шаблон "User Journey Map"
3. Добавить блоки для каждого экрана
4. Соединить потоки
5. Поделиться ссылкой или экспортировать

## 🎨 Цветовая Схема

### Основные цвета:
- **Синий (#2563EB)** — основной, технологии
- **Зелёный (#10B981)** — деньги, успех
- **Красный (#EF4444)** — удаление, отмена
- **Жёлтый (#F59E0B)** — предупреждения
- **Серый (#6B7280)** — второстепенная информация

### Применение:
- **Действия пользователя** → Синий
- **Финансовые операции** → Зелёный
- **Админ-функции** → Фиолетовый (#8B5CF6)
- **Автоматизация** → Оранжевый (#F97316)

## 🚀 Пошаговая Инструкция для Gamma

### Шаг 1: Подготовка контента
1. Открой `GAMMA_PRESENTATION_CONTENT.md`
2. Скопируй весь текст
3. Зайди на gamma.app

### Шаг 2: Создание презентации
1. Нажми "New presentation"
2. Выбери "Paste in text"
3. Вставь скопированный текст
4. Нажми "Generate"

### Шаг 3: AI создаст слайды
- Подожди 1-2 минуты
- AI автоматически создаст дизайн
- Разместит текст и элементы

### Шаг 4: Редактирование
1. Просмотри каждый слайд
2. Добавь скриншоты бота (если есть)
3. Настрой цвета под бренд
4. Добавь логотип

### Шаг 5: Финализация
1. Проверь орфографию
2. Убедись в логике повествования
3. Добавь контакты на последний слайд
4. Экспортируй в PDF или поделись ссылкой
