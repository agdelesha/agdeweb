## 🔐 ВОРОНКА ПОКУПКИ

### Путь 1: Новый пользователь

```
Получить доступ
    │
    ├─→ Проверка пароля (если включено)
    │       │
    │       └─→ Ввод пароля → Проверка
    │
    ├─→ Проверка подписки на канал (если включено)
    │       │
    │       └─→ Кнопка "Подписаться" → Проверка подписки
    │
    ├─→ Запрос телефона (если включено)
    │       │
    │       └─→ Кнопка "Поделиться" или "Пропустить"
    │
    └─→ Выбор тарифа
            │
            ├─→ 7 дней — бесплатно (пробный)
            │       │
            │       └─→ Создание конфига → Отправка пользователю
            │
            └─→ Платные тарифы
                    │
                    ├─→ 30 дней — 100₽
                    ├─→ 90 дней — 200₽
                    └─→ 180 дней — 300₽
                            │
                            └─→ Реквизиты для оплаты
                                    │
                                    └─→ Отправка чека
                                            │
                                            └─→ Проверка админом
                                                    │
                                                    ├─→ Одобрено → Создание конфига
                                                    └─→ Отклонено → Уведомление
```

### Путь 2: Существующий пользователь

```
Получить доступ
    │
    └─→ Проверка активной подписки
            │
            ├─→ Есть подписка → "У тебя уже есть подписка"
            └─→ Нет подписки → Выбор тарифа (без пробного)
```

## 🎯 КЛЮЧЕВЫЕ ОСОБЕННОСТИ

### 1. Мультибот-система
- Несколько ботов на одном сервере
- Индивидуальные настройки для каждого
- Добавление через админ-панель

### 2. Мультисервер
- Автоматическое распределение клиентов
- Балансировка нагрузки
- Серверы в разных странах

### 3. Реферальная программа
- 50% скидка для приглашённых
- Процент от платежей рефереру
- Вывод средств через админа

### 4. Безопасность
- Пароль для доступа (опционально)
- Обязательная подписка на канал (опционально)
- Запрос телефона (опционально)
- Лимиты конфигов

### 5. Автоматизация
- Автоматическое создание конфигов
- Уведомления об окончании подписки
- Автоматическое отключение
- Мониторинг активности

## 🎨 РЕКОМЕНДАЦИИ ДЛЯ ИНФОГРАФИКИ

### Инструменты для создания:
1. **Figma** — профессиональный дизайн
2. **Canva** — простой и быстрый
3. **Draw.io** — для блок-схем
4. **Miro** — для интерактивных карт
5. **Gamma.app** — для презентаций с AI

### Стиль:
- Используй иконки для каждой кнопки
- Цветовое кодирование (зелёный = действие, синий = информация, красный = удаление)
- Стрелки для показа потока
- Группировка связанных функций

### Структура слайдов для Gamma:

**Слайд 1:** Обзор проекта
